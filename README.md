- `combination.cpp`
	- Combination of modulo

- `divisors.cpp`

	- 2D points utility class

- `lazy_segment_tree.cpp`

- `lca.cpp`
	- Lowest common ancestor

- `mod.cpp`
	- Modular calculus utility class
- `prime.cpp`
	- Prime factorization
- `scc.cpp`
	- Strongly connected component
- `segment_tree.cpp`
	- Segment tree
- `union_find.cpp`
	- Union find tree
- `z_algo.cpp`
	- Z algorithm (substring matching)
